PRISMELEC - fichiers renommes

Structure preparee pour reconstruction GitHub Pages.

Dossiers principaux :
- audio : chansons MP3
- srt : fichiers de paroles synchronisees
- textes : textes de chansons et sources ODT
- images : pochettes, planches de presentation, documents visuels
- situations/vierges : planches de situations vierges
- situations/remplies : planches de situations remplies
- support : documents d'animation pedagogique
- html : ancien index conserve sous index-original.html
- archives : fichiers originaux utiles conserves a part

Le fichier manifest-renommage.csv donne la correspondance ancien nom -> nouveau nom.

Version prete GitHub : les PDF de synthese ont ete optimises pour reduire le poids. Les archives doublons ont ete exclues.
